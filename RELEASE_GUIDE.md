# How to Create a GitHub Release

Follow these steps to upload the APK as a GitHub Release:

### 1. Go to GitHub Releases Page
Visit: https://github.com/CodesRahul96/Exclusive-TV-Updates/releases

### 2. Click "Create a new release"

### 3. Fill in Release Details

**Choose a tag:**
- Click "Choose a tag"
- Type: `v1.0.31`
- Click "Create new tag: v1.0.31 on publish"

**Release title:**
```
ExclusiveTV v1.0.31
```

**Description (optional):**
```markdown
## ExclusiveTV v1.0.31

### What's New
- Dual-source update system with automatic fallback
- Performance improvements and bug fixes
- Enhanced stability

### Download
Download the APK below and install on your Android TV device.
```

### 4. Upload APK File

- Click "Attach binaries by dropping them here or selecting them"
- Select: `ExclusiveTV-v1.0.31.apk` (from this directory)
- Wait for upload to complete

### 5. Publish Release

- Click "Publish release"
